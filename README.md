# Python aplikácia – Rozdiely medzi číslami

Tento projekt je jednoduchá grafická aplikácia vytvorená v jazyku **Python** pomocou knižnice **Tkinter**.  
Aplikácia generuje náhodné čísla, vypočíta rozdiely medzi susednými hodnotami a ku každému rozdielu pripočíta konštantu **K**.  
Výsledky zobrazí v dvoch zoznamoch a zároveň vykreslí graf pomocou **Matplotlib**.


## Funkcionalita programu

- používateľ zadá:
  - **N** – počet generovaných čísel
  - **K** – hodnotu, ktorá sa pripočíta ku každému rozdielu
- program vygeneruje N náhodných čísel
- vypočíta rozdiely medzi susednými číslami
- ku každému rozdielu pripočíta hodnotu K
- zobrazí:
  - pôvodné čísla
  - rozdiely + K
- vykreslí graf upravených rozdielov

## Použité technológie a knižnice

- **Python 3**
- **Tkinter** (GUI)
- **Matplotlib** (graf)
- **random** (generovanie čísel)
- **PyInstaller** (vytvorenie .exe súboru)
